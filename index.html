Spotify・Steam・Last.fm連携から、画像・統計グリッド・コレクション・進捗バーまで。 Discord Profile Widgets V2 用の Node.js / Google Apps Script テンプレートを組み立てるための、 4つのページに分かれたツールです。入力した資格情報はどこにも送信されません。

概要

PATCHBAYでできること

Discordの実験的機能「Profile Widgets V2」用のウィジェットを、コードを書かずに組み立てて、実行用のコードだけを持ち帰れるツールです。

01 ブラウザだけで完結 Application ID、Bot Token、Spotify/Steam/Last.fmの資格情報はすべてこのページ内のJavaScriptで処理され、外部サーバーには一切送信されません。
02 7種のプリセット＋詳細ビルダー Spotify・Steam・Last.fm連携のプリセットや、画像・コレクション・進捗バーなどの構成をすぐに使えます。もちろん、まっさらな状態からすべて手動で組み立てることもできます。
03 Node.js / GASの2形式で出力 自分のPCやサーバーで動かすNode.jsスクリプトと、Googleアカウントだけで定期実行できるGoogle Apps Scriptの、どちらの形式でもコードを生成できます。
仕組み

なぜ安全に使えるか

DWidgetsのような第三者サービスは便利な反面、自分のBot Tokenをそのサービスのサーバーに預ける必要があります。 PATCHBAYは真逆のアプローチを取ります。

サーバーを持たない、という設計

このサイトはGitHub Pagesなどの静的ホスティング上で動作する、ただのHTMLファイル群です。フォームに入力した値は、 「Discordへ送るためのJavaScriptコードの文字列」を組み立てる材料としてのみブラウザ内で使われ、どこにも送信・保存されません。 生成されたコードをコピーまたはダウンロードしたら、実際にDiscordのAPIを呼び出すのは、あなた自身のPCやサーバー、 またはあなたのGoogleアカウント上のApps Scriptです。つまりPATCHBAYを閉じてしまっても、あなたの資格情報がどこかに残ることはありません。
