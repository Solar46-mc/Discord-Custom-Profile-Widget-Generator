◱ Widget Bench

Discord Profile Widgets V2 の構成を組み立てて、貼り付けるだけの状態にする非公式ツール

注意： Widgets V2 はDiscordの実験的・未文書化の機能です。仕様変更で動かなくなる可能性、規約上グレーな挙動（DevToolsコンソールでの内部モジュール操作を含む）を伴います。自己責任で利用してください。

ウィジェット上部のレイアウト

Discordがウィジェットの最上部をどう描画するかを選びます。

PNG / JPG / GIFに対応。ヒーロー表示では最初の約28文字より後は「…」で省略されます。

ミニプロフィールのレイアウト

ウィジェットにカーソルを合わせたときにアプリアイコン横へ表示される内容です。

ウィジェット下部のレイアウト

タイトルの下に表示する追加ブロックです。不要なら「なし」を選べます。

データの取得元

値を固定にするか、botに定期更新させるかを選びます。「User Data」を選んだ項目はbotがAPI経由で更新できます。

書き出し

以下をそれぞれコピーして使ってください。上から順番に進めれば設定が完了します。

① Developer Portal で作成するフィールド一覧

Developer Portal のウィジェット編集画面で、ここに書かれた名前・タイプと完全に一致するフィールドを先に作成してください（名前は大文字小文字も含めて一致させる必要があります）。

③ アプリをプロフィールの候補に追加するコンソールスクリプト

          
        

        
Discord（ブラウザ版推奨）で Ctrl+Shift+I → Console タブに貼り付けて実行します。実行後 allow pasting の入力を求められる場合はそれも入力してください。あわせて実験フラグ 2026-03-application-widget-v2-renderer を Variant 1 にしておくと表示が安定します。

        

      


      

        

          
④ 初回データ送信コマンド（curl / PowerShell）

          
          
        

        
APPLICATION_ID / USER_ID / BOT_TOKEN をご自身の値に置き換えてください。BOT_TOKEN は絶対に人に共有・コミットしないでください。
